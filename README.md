# Shrunk 3D Business Organization System

A comprehensive file and asset management system for your 3D shrunk business with GoHighLevel integration capabilities.

## Project Structure

```
shrunk-3d-business/
├── assets/                    # All business assets
│   ├── images/               # Product images and media
│   ├── 3d-models/           # 3D model files
│   ├── documents/           # Business documents
│   └── templates/           # Design templates
├── clients/                  # Client-specific folders
├── projects/                # Active projects
├── marketing/               # Marketing materials
├── ghl-integration/         # GoHighLevel integration
├── tools/                   # Management scripts
└── config/                  # Configuration files

## Features

- **Automated File Organization**: Smart categorization based on file types
- **Client Management**: Dedicated folders for each client
- **Project Tracking**: Organized project workflows
- **GoHighLevel Integration**: API integration for CRM sync
- **Backup System**: Automated backup to AI Drive
- **Search & Discovery**: Easy file finding tools

## Quick Start

1. Run `npm install` to install dependencies
2. Configure your GoHighLevel API credentials in `config/ghl-config.json`
3. Use `npm run organize` to auto-organize existing files
4. Use `npm run backup` to create backups
